# hotelspace

개발 기간 : 2020.09 ~ 2020.10

개발 환경 : window, eclipse, spring 

개발 인원 : 7명

요약 : 호텔 예약 웹 페이지

기능 
1) Client)날짜 별 호텔 검색, 예약 및 결재
2) Client) FAQ , QnA
3) Client) Login, my page
4) Admin)통계정보 
5) Admin)회원 관리
5) Admin)페이지 관리

맡은 기능 : 로그인(kakao login api), 관리자 메인 통계 페이지, 관리자 회원 관리 페이지, 
 클라이언트 메인 페이지, 게시판 및 목록 페이징 처리, 오류 처리
